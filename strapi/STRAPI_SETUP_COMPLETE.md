### 5. 📍 Место (Place)
**Назначение:** Основная коллекция достопримечательностей  
**Статус:** Готово к заполнению

**Структура:**
- `name` - Название места (обязательно)
- `slug` - URL-идентификатор (авто)
- `image` - Фотография (single image)
- `history` - История (richtext)
- `address` - Адрес
- `latitude` - Широта (decimal)
- `longitude` - Долгота (decimal)
- `working_hours` - Часы работы
- `phone` - Телефон
- `website` - Сайт
- `is_active` - Активность
- **Связи:**
  - `area` - Район (manyToOne)
  - `categories` - Категории (manyToMany)
  - `tags` - Теги (manyToMany)
  - `routes` - Маршруты (manyToMany)

## 🎯 Примеры API запросов

### Получить все места с полной информацией
```bash
curl "http://localhost:1337/api/places?populate=*"
```

### Получить места по району (например, Эльбрусский)
```bash
curl "http://localhost:1337/api/places?filters[area][id]=10&populate=*"
```

### Получить места по категории (например, Экотуризм)
```bash
curl "http://localhost:1337/api/places?filters[categories][id]=1&populate=*"
```

### Получить места по тегу (например, Водопад)
```bash
curl "http://localhost:1337/api/places?filters[tags][id]=2&populate=*"
```

### Получить маршруты определенного типа (например, Пеший)
```bash
curl "http://localhost:1337/api/routes?filters[route_type][id]=1&populate=*"
```

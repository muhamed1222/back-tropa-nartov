# 🇷🇺 Руководство: Русские названия в Strapi

## ⚠️ Важно

В **Strapi v4 нельзя перевести названия полей** (`name`, `address`, `short_description` и т.д.) через JSON схему в админке.

**Причина:** Названия полей берутся из **API ID**, который должен быть на английском для совместимости с REST API.

---

## ✅ Что УЖЕ переведено

### Collection Types (Контент-типы)

Все контент-типы имеют русские названия в админке:

| Type | **Тип** | ✅ |
| Area | **Область** | ✅ |
| Image | **Изображение** | ✅ |

## 📋 Названия полей - Ограничение Strapi

### Почему нельзя перевести

**Названия полей** (`name`, `address`, `working_hours` и т.д.) в админке Strapi берутся из **API ID полей**, который используется в REST API:

```json
GET /api/places?populate=*
{
  "data": [{
    "attributes": {
      "name": "...",        // ← Это API ID поля
      "address": "...",     // ← Это API ID поля
      "working_hours": "..." // ← Это API ID поля
    }
  }]
}
```

Если изменить API ID на русский, API сломается.

## 💡 Решения

### Вариант 1: Использовать понятные английские названия (рекомендуется) ✅

Текущие названия полей уже понятны и стандартны:

| Поле | Что означает | Для редактора |
|------|--------------|---------------|
| `name` | Название | **Название** места |
| `short_description` | Краткое описание | **Краткое описание** (для превью) |
| `address` | Адрес | **Адрес** |
| `working_hours` | Часы работы | **Часы работы** |
| `contacts_phone` | Телефон | **Телефон** |
| `contacts_website` | Веб-сайт | **Веб-сайт** |
| `latitude` | Широта | **Широта** (для карты) |
| `longitude` | Долгота | **Долгота** (для карты) |
| `history` | История | **История** места |
| `images` | Изображения | **Изображения** |

**Преимущество:** ✅ Работает с API без изменений, понятно редакторам.

---

### Вариант 2: Добавить описания через Content-Type Builder

Можно добавить **описания** к полям в админке Strapi:

1. Откройте **Content-Type Builder**
2. Выберите **Место** → **name**
3. Добавьте **Description**: "Название места"
4. Это будет подсказкой в админке

**Недостаток:** ⚠️ Описание - это не название поля.

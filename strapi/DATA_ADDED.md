## 🔍 Проверка данных

### Через Strapi админ-панель:
1. Откройте: http://localhost:1337/admin
2. Перейдите в **Content Manager**
3. Выберите **Place** или **Route** для просмотра добавленных записей

### Через API:
```bash
# Получить все места
curl http://localhost:1337/api/places

# Получить все маршруты
curl http://localhost:1337/api/routes

# Получить конкретное место
curl http://localhost:1337/api/places/1
```

### Через PostgreSQL:
```bash
docker-compose exec postgres psql -U postgres -d tropa_nartov -c "SELECT id, name FROM places;"
docker-compose exec postgres psql -U postgres -d tropa_nartov -c "SELECT id, name FROM routes;"
```

## 📝 Примечания

- Данные добавлены напрямую в PostgreSQL

# 🚀 Быстрая инструкция: Как добавить место в Strapi

## 📍 Шаг 1: Откройте админ-панель

```
http://localhost:1337/admin
```

Войдите с учетными данными администратора.

---

## 📝 Шаг 2: Создание нового места

### 2.1. Перейдите в Content Manager

1. В левом меню нажмите **"Content Manager"**
2. Выберите **"Place"** из списка

### 2.2. Создайте новую запись

1. Нажмите кнопку **"+ Create new entry"** (вверху справа)

---

## ✍️ Шаг 3: Заполнение формы

### Обязательные поля ⚠️ (Required):

| Поле | Пример | Описание |
|------|--------|----------|
| **name** | `Парк Атажукинский` | Название места |
| **description** | `Крупнейший парк...` | Подробное описание (Rich Text) |
| **address** | `г. Нальчик, ул. Толстого, 2` | Полный адрес |
| **latitude** | `43.4925` | Широта (decimal) |
| **longitude** | `43.6123` | Долгота (decimal) |
| **area_id** | `1` | ID района (число) |

### Необязательные поля:

| Поле | Пример | Описание |
|------|--------|----------|
| **type** | `парк` | Тип места (кафе, музей, парк и т.д.) |
| **overview** | `Площадь 250 гектаров...` | Краткий обзор |
| **history** | `Заложен в 1847 году...` | Историческая справка |
| **hours** | `06:00-24:00` | Часы работы |
| **weekend** | `Без выходных` | Выходные дни |
| **entry** | `Свободный` | Тип входа (Свободный/Платный) |
| **contacts** | `+78662225566` | Телефон |
| **contacts_email** | `park@nalchik.ru` | Email |
| **rating** | `4.8` | Рейтинг (0-5) |
| **is_active** | ✅ | Активно ли место |

### Связи (Relations):

- **area** - Выберите район из списка
- **categories** - Выберите категории (можно несколько)
- **tags** - Выберите теги (можно несколько)
- **images** - Добавьте изображения (можно позже)

---

## 💾 Шаг 4: Сохранение и публикация

### Важно! ⚠️

1. **Save** (Сохранить) - сохраняет как **черновик** (недоступно через API)
2. **Publish** (Опубликовать) - делает доступным через **API**

**Всегда нажимайте "Publish" после сохранения!**

---

## 📋 Пример заполнения

### Пример: Парк

```

Тип: парк
Описание: Крупнейший парк на Северном Кавказе с богатой растительностью
Обзор: Площадь 250 гектаров, аллеи, фонтаны, озера

Выходные: Без выходных

Широта: 43.4925
Долгота: 43.6123
Рейтинг: 4.8
Район: Приэльбрусье (выбрать из списка)
```

---

## 🖼️ Добавление изображений

### Способ 1: Прямо в форме места

1. Прокрутите вниз до поля **"images"**
2. Нажмите **"Add an entry"**
3. Нажмите **"Upload"** или перетащите файлы
4. Выберите изображения
5. Сохраните

### Способ 2: Через Media Library

1. Сначала загрузите в **Media Library**
2. Затем выберите из библиотеки в форме места

---

## 🔗 Связывание с другими данными

### Добавить категории:

1. Найдите поле **"categories"**
2. Нажмите **"Add an entry"**
3. Выберите категории (можно несколько)
4. Нажмите **"Finish"**

### Добавить теги:

1. Найдите поле **"tags"**
2. Нажмите **"Add an entry"**
3. Выберите теги (можно несколько)
4. Нажмите **"Finish"**

### Выбрать район:

1. Найдите поле **"area"**
2. Нажмите **"Select an entry"**
3. Выберите район из списка
4. Нажмите **"Finish"**

---

## ✅ Чеклист

Перед публикацией проверьте:

- [ ] Заполнены все обязательные поля
- [ ] Координаты указаны правильно (decimal числа)
- [ ] Выбран район (area)
- [ ] Описание заполнено
- [ ] Нажата кнопка **"Publish"** (не только Save!)

---

## 📡 Проверка через API

После публикации проверьте:

```bash
# Получить все места
curl http://localhost:1337/api/places

# Получить конкретное место
curl http://localhost:1337/api/places/1
```

---

## ⚙️ Если API не работает

Настройте права доступа:

1. **Settings** → **Users & Permissions plugin** → **Roles** → **Public**
2. Найдите **Place**
3. Включите:
   - ✅ **find** (GET /api/places)
   - ✅ **findOne** (GET /api/places/:id)
4. Сохраните

---

## 🎯 Полезные советы

1. **Всегда публикуйте** - только опубликованные записи видны в API
2. **Используйте Rich Text** - для красивого форматирования описаний
3. **Проверяйте координаты** - используйте Google Maps для точности
4. **Добавляйте изображения** - визуальный контент важен
5. **Используйте категории и теги** - для лучшей организации

---

## 🆘 Частые проблемы

### Место не видно в API
- ✅ Проверьте, что нажали **"Publish"** (не только Save)
- ✅ Проверьте права доступа для Public роли

### Ошибка при сохранении
- ✅ Проверьте обязательные поля (name, description, address, latitude, longitude, area_id)
- ✅ Проверьте формат координат (числа с точкой, например: 43.4925)

### Не могу выбрать район
- ✅ Сначала создайте район в Content Manager → Area

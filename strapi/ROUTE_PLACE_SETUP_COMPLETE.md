## Важные замечания

### Использование компонента stops

Компонент `route-stop` позволяет:
- Создавать упорядоченный список остановок в маршруте
- Каждая остановка содержит:
  - `place` - связь с моделью Place (oneToOne)
  - `order` - порядковый номер остановки в маршруте (integer)

### Автоматические расчеты

При сохранении маршрута (создании или обновлении):
1. Система автоматически валидирует остановки
2. Загружает координаты всех мест из остановок
3. Рассчитывает расстояние по формуле Хаверсина между последовательными точками
4. Определяет длительность маршрута на основе типа (пеший/вело/авто)
5. Сохраняет количество остановок

### Формула Хаверсина

Используется для расчета расстояния между двумя точками на сфере (Земле):
- Учитывает кривизну Земли
- Радиус Земли = 6371 км
- Результат в километрах с точностью до сотых

### Типы маршрутов и скорости

Система автоматически определяет среднюю скорость на основе названия типа:
- **Пеший** (слова: "пеш", "ходьба", "walking") → 3 км/ч
- **Велосипедный** (слова: "вело", "велосипед", "cycling") → 12 км/ч
- **Автомобильный** (слова: "авто", "машин", "driving") → 50 км/ч
- **По умолчанию** → 3 км/ч

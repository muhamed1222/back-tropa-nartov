# 📝 Как добавлять МЕСТО - Новая структура

## 🚀 Быстрый старт

1. **Откройте админ-панель:** `http://localhost:1337/admin`
2. **Content Manager** → **Место** → **+ Create new entry**

---

## 📋 Заполнение полей

### 🔹 1. Основные поля (вверху формы)

| Поле | Обязательное | Пример | Описание |
|------|--------------|--------|----------|
| **name** | ✅ Да | `Парк Атажукинский` | Название места |
| **slug** | ❌ Нет | `park-atazhukinskiy` | URL-фрагмент (автогенерация) |
| **short_description** | ❌ Нет | `Крупнейший парк Нальчика` | Краткое описание для превью (до 500 символов) |
| **type** | ❌ Нет | Выбрать из списка | Тип места (Парк, Музей, Кафе и т.д.) |
| **images** | ❌ Нет | Загрузить файлы | Изображения (Media Library) |

**Как загрузить изображения:**
- Нажмите на поле "images"
- Выберите "Upload files" или "Choose existing"
- Можно выбрать несколько изображений

---

### 🔹 2. История (вкладка или раздел)

| Поле | Обязательное | Пример | Описание |
|------|--------------|--------|----------|
| **history** | ❌ Нет | `Заложен в 1847 году...` | Историческая информация (Rich Text) |

**Совет:** Используйте форматирование текста (жирный, курсив, списки).

---

### 🔹 3. Обзор (вкладка или раздел)

| Поле | Обязательное | Пример | Описание |
|------|--------------|--------|----------|
| **address** | ❌ Нет | `г. Нальчик, ул. Толстого, 2` | Полный адрес |
| **working_hours** | ❌ Нет | `06:00-24:00, без выходных` | Часы работы (можно добавить выходные) |
| **contacts_phone** | ❌ Нет | `+78662225566` | Телефон |
| **contacts_website** | ❌ Нет | `https://park-nalchik.ru` | Веб-сайт |

---

### 🔹 4. Геоданные (обязательно!)

| Поле | Обязательное | Пример | Описание |
|------|--------------|--------|----------|
| **latitude** | ✅ Да | `43.4925` | Широта (для карты) |
| **longitude** | ✅ Да | `43.6123` | Долгота (для карты) |

**Как найти координаты:**
1. Откройте Google Maps
2. Найдите место на карте
3. Правый клик → "Что здесь?"
4. Скопируйте координаты

---

### 🔹 5. Связи

| Поле | Обязательное | Описание |
|------|--------------|----------|
| **area** | ❌ Нет | Область/район (выберите из списка) |
| **categories** | ❌ Нет | Категории (можно выбрать несколько) |
| **tags** | ❌ Нет | Теги (можно выбрать несколько) |
| **reviews** | ❌ Нет | Отзывы (создаются отдельно) |

---

### 🔹 6. Системные поля

| Поле | По умолчанию | Описание |
|------|--------------|----------|
| **is_active** | ✅ `true` | Активно ли место (галочка) |

## ⭐ Важно: Рейтинг НЕ вводится вручную!

### ❌ НЕ заполняйте:
- Рейтинг **НЕ** вводится в поле Place
- Рейтинг рассчитывается автоматически из отзывов

### ✅ Как добавляется рейтинг:

1. **Создайте отзыв:**
   - **Content Manager** → **Отзыв** → **+ Create new entry**

2. **Заполните отзыв:**
   - **place** - выберите место из списка
   - **user_name** - имя пользователя (например, `Магомед`)
   - **rating** - оценка от 1 до 5 ⭐
   - **text** - текст отзыва (опционально)
   - **date** - дата (автоматически устанавливается)

3. **Сохраните и опубликуйте отзыв**

4. **Рейтинг считается автоматически:**
   - Приложение получает все отзывы для места
   - Считает среднее: `(sum of ratings) / (count of reviews)`
   - Показывает: ⭐ 4.8

## 📝 Пошаговый пример создания места

### Шаг 1: Подготовка

✅ Убедитесь, что созданы:
- **Область** (например, "Нальчик")
- **Тип места** (например, "Парк")
- **Категории** (например, "Достопримечательности")

### Шаг 2: Создание места

1. **Content Manager** → **Место** → **+ Create new entry**

2. **Заполните основные поля:**
   ```
   name: Парк Атажукинский
   short_description: Крупнейший парк Нальчика площадью 250 гектаров
   type: [Выбрать из списка: Парк]
   ```

3. **Загрузите изображения:**
   - Нажмите на "images"
   - Выберите "Upload files"
   - Загрузите 3-5 фотографий парка

4. **Заполните историю:**
   ```
   history: Заложен в 1847 году по инициативе... (Rich Text)
   ```

5. **Заполните обзор:**
   ```
   address: г. Нальчик, ул. Толстого, 2
   working_hours: 06:00-24:00, без выходных
   contacts_phone: +78662225566
   contacts_website: https://park-nalchik.ru
   ```

6. **Укажите координаты:**
   ```
   latitude: 43.4925
   longitude: 43.6123
   ```

7. **Выберите связи:**
   - area: [Выбрать: Нальчик]
   - categories: [Выбрать: Достопримечательности, Развлечения]
   - tags: [Выбрать: парк, нальчик, природа]

8. **Сохраните:**
   - Нажмите **"Save"** (Сохранить)
   - Нажмите **"Publish"** (Опубликовать)

### Шаг 3: Добавление отзывов (для рейтинга)

1. **Content Manager** → **Отзыв** → **+ Create new entry**

2. **Заполните отзыв:**
   ```
   place: [Выбрать: Парк Атажукинский]
   user_name: Магомед
   rating: 5
   text: Отличный парк! Очень красиво и ухожено.
   date: [Автоматически устанавливается]
   ```

3. **Save** → **Publish**

4. **Добавьте еще 2-3 отзыва** для среднего рейтинга

## 🌐 Доступ через API

После публикации место доступно:

```bash
# Получить все места
GET http://localhost:1337/api/places?populate=*

# Получить место с отзывами (для расчета рейтинга)
GET http://localhost:1337/api/places/:id?populate[reviews][fields][0]=rating&populate[reviews][fields][1]=user_name&populate[reviews][fields][2]=text&populate[reviews][fields][3]=date
```

**Рейтинг считается на стороне приложения:**
```javascript
const place = await api.getPlace(id);
const rating = place.reviews.reduce((sum, r) => sum + r.rating, 0) / place.reviews.length;
// Показываем: ⭐ 4.8
```

---

## 💡 Советы

1. **short_description** делайте коротким (до 500 символов) - это для превью в списке мест
2. **history** можно делать длинным с форматированием (Rich Text)
3. **working_hours** можно включить информацию о выходных: `06:00-24:00, без выходных`
4. **Изображения** загружайте несколько (3-5 штук) - они показываются в слайдере
5. **Всегда публикуйте** записи (Publish), иначе они не доступны через API
6. **Рейтинг** добавляется через отзывы, не вручную!

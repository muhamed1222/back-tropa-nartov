# 📝 Добавление мест через админ-панель Strapi

Поскольку автоматическое добавление через API требует настройки прав доступа, вот инструкция по ручному добавлению мест через админ-панель.

## 🚀 Быстрое добавление

1. **Откройте админ-панель:** `http://localhost:1337/admin`
2. **Content Manager** → **Место** → **+ Create new entry**
3. Скопируйте данные из таблицы ниже и заполните форму

## ✅ После добавления

1. **Сохраните** каждое место (кнопка "Save")
2. **Опубликуйте** каждое место (кнопка "Publish")
3. Проверьте в списке мест, что все добавлено

---

## 🔍 Проверка через API

После публикации места будут доступны через API:

```bash
GET http://localhost:1337/api/places?populate=*
```

# Strapi Админ-панель для Тропа Нартов

Это Strapi CMS для управления контентом проекта "Тропа Нартов".

## Доступ

- **Админ-панель**: http://localhost:1337/admin
- **API**: http://localhost:1337/api

## Первый запуск

1. Strapi автоматически создастся при первом запуске
2. Откройте http://localhost:1337/admin
3. Создайте первого администратора
4. Начните создавать Content Types для ваших данных

## Content Types

Рекомендуется создать следующие Content Types в Strapi:

- **Place** - Места и достопримечательности
- **Route** - Туристические маршруты
- **Review** - Отзывы
- **Category** - Категории
- **Tag** - Теги
- **Article** - Статьи

## Подключение к БД

Strapi использует ту же PostgreSQL БД, что и основной бэкенд.
Все данные хранятся в отдельных таблицах с префиксом `strapi_`.

## Переменные окружения

Настройки БД берутся из `.env` файла основного проекта.

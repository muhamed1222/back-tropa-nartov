# 📝 Как добавлять контент в Strapi - Подробное руководство

## 🚀 Быстрый старт

### 1. Откройте админ-панель Strapi

```
http://localhost:1337/admin
```

Войдите с учетными данными администратора.

---

## 📍 Добавление МЕСТА (Place)

### Шаг 1: Откройте Content Manager

1. В левом меню нажмите **"Content Manager"** (Диспетчер контента)
2. Выберите **"Место"** из списка Collection Types

### Шаг 2: Создайте новое место

1. Нажмите кнопку **"+ Create new entry"** (вверху справа)
2. Откроется форма для заполнения

---

## 📋 Заполнение полей для МЕСТА

### ⚠️ ОБЯЗАТЕЛЬНЫЕ ПОЛЯ (Required)

| Поле | Русское название | Тип | Пример | Описание |
|------|------------------|-----|--------|----------|
| **name** | Название | Текст (200 символов) | `Парк Атажукинский` | Название места |
| **description** | Описание | Rich Text | `Крупнейший парк Нальчика площадью...` | Подробное описание места (с форматированием) |
| **address** | Адрес | Текст (500 символов) | `г. Нальчик, ул. Толстого, 2` | Полный адрес места |
| **latitude** | Широта | Число (decimal) | `43.4925` | Координата широты (для карты) |
| **longitude** | Долгота | Число (decimal) | `43.6123` | Координата долготы (для карты) |
| **area** | Область | Связь (Many-to-One) | `Нальчик` | Выберите область из списка |

---

### 📝 НЕОБЯЗАТЕЛЬНЫЕ ПОЛЯ

#### Основная информация

| Поле | Русское название | Тип | Пример | Описание |
|------|------------------|-----|--------|----------|
| **type** | Тип места | Связь (Many-to-One) | `Парк` | Выберите тип из списка (должен быть создан заранее) |
| **type_name** | Название типа | Текст (100 символов) | `парк` | Альтернативное поле для типа (текст) |
| **overview** | Обзор | Rich Text | `Площадь 250 гектаров, открыт круглосуточно` | Краткий обзор места |
| **history** | История | Rich Text | `Заложен в 1847 году по инициативе...` | Историческая справка о месте |
| **rating** | Рейтинг | Число (decimal) | `4.8` | Рейтинг места (0-5) |

#### Контактная информация

| Поле | Русское название | Тип | Пример |
|------|------------------|-----|--------|
| **hours** | Часы работы | Текст (200 символов) | `06:00-24:00` |
| **opening_hours** | Часы открытия | Текст (100 символов) | `С 6 утра до полуночи` |
| **weekend** | Выходные | Текст (100 символов) | `Без выходных` |
| **entry** | Вход | Текст (100 символов) | `Свободный` или `Платный` |
| **contacts** | Контакты | Текст (200 символов) | `+78662225566` |
| **contacts_email** | Email | Email | `park@nalchik.ru` |

#### Связи и теги

| Поле | Русское название | Тип | Описание |
|------|------------------|-----|----------|
| **categories** | Категории | Связь (Many-to-Many) | Выберите категории (можно несколько) |
| **tags** | Теги | Связь (Many-to-Many) | Выберите теги (можно несколько) |
| **images** | Изображения | Связь (One-to-Many) | Изображения добавятся автоматически после создания |
| **is_active** | Активно | Boolean | Галочка (по умолчанию включена) |

## 🎯 Пошаговый процесс создания места

### 1️⃣ Подготовка данных

**Перед созданием места убедитесь, что у вас есть:**

- ✅ **Область** создана (например, "Нальчик", "Эльбрус")
- ✅ **Тип места** создан (например, "Парк", "Музей", "Кафе")
- ✅ **Категории** созданы (например, "Достопримечательности", "Развлечения")
- ✅ **Теги** созданы (опционально)

### 2️⃣ Создание места

1. **Content Manager** → **Место** → **+ Create new entry**

2. **Заполните обязательные поля:**
   - Название: `Парк Атажукинский`
   - Описание: Введите подробное описание (можно использовать форматирование)
   - Адрес: `г. Нальчик, ул. Толстого, 2`
   - Широта: `43.4925`
   - Долгота: `43.6123`
   - Область: Выберите из выпадающего списка (например, "Нальчик")

3. **Заполните необязательные поля:**
   - Тип: Выберите из списка (например, "Парк")
   - Обзор: Краткое описание
   - История: Историческая информация
   - Часы работы: `06:00-24:00`
   - Контакты: `+78662225566`
   - Email: `park@nalchik.ru`
   - Рейтинг: `4.8`

4. **Выберите категории и теги:**
   - Нажмите на поле "Категории"
   - Выберите нужные категории из списка (можно несколько)
   - Аналогично с тегами

5. **Сохраните:**
   - Нажмите **"Save"** (Сохранить) вверху справа
   - Запись будет сохранена как **Draft** (черновик)

6. **Опубликуйте:**
   - Нажмите **"Publish"** (Опубликовать)
   - Теперь место доступно через API

## 📸 Добавление изображений

### Вариант 1: Через тип "Изображение"

1. **Content Manager** → **Изображение** → **+ Create new entry**
2. Заполните:
   - **url**: `https://example.com/image.jpg`
   - **alt_text**: `Парк Атажукинский - главный вход`
   - **place**: Выберите созданное место из списка
3. **Save** → **Publish**

## 🔄 Редактирование места

1. **Content Manager** → **Место**
2. Найдите нужное место в списке
3. Кликните на название или нажмите **"Edit"** (Редактировать)
4. Внесите изменения
5. **Save** → **Publish** (если нужно опубликовать изменения)

---

## 📊 Draft & Publish (Черновики и публикация)

Strapi использует систему черновиков:

- **Draft** (Черновик) - запись сохранена, но не опубликована (недоступна через API)
- **Published** (Опубликовано) - запись доступна через API

**Важно:** После сохранения всегда нажимайте **"Publish"**, чтобы запись была доступна в приложении.

---

## 🌐 Доступ через API

После публикации место доступно по адресу:

```
GET http://localhost:1337/api/places
GET http://localhost:1337/api/places/:id
```

Для получения с связанными данными:

```
GET http://localhost:1337/api/places?populate=*
GET http://localhost:1337/api/places/:id?populate=*
```

## 💡 Советы

1. **Координаты** можно найти на Google Maps:

   - Правый клик → "Что здесь?"
   - Скопируйте координаты

2. **Описание** делайте подробным - это улучшит SEO и UX

3. **Изображения** добавляйте через отдельный тип "Изображение" и связывайте с местом

4. **Категории и теги** помогают фильтровать места в приложении

5. **Всегда публикуйте** записи после создания, иначе они не будут доступны через API

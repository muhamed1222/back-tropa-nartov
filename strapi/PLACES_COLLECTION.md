# 📍 Коллекция "Место"

## Структура полей

### 📝 Основная информация
- **name** (string, required) - Название места (максимум 200 символов)
- **slug** (uid, required) - URL-идентификатор (генерируется автоматически из названия)

### 🖼️ Медиа
- **image** (media, single) - Фотография места (только изображения)

### 📖 Контент
- **history** (richtext) - История места (поддерживает форматирование)

### 📍 Местоположение
- **address** (string) - Адрес (максимум 500 символов)
- **latitude** (decimal) - Широта
- **longitude** (decimal) - Долгота

### 📞 Контакты
- **working_hours** (text) - Часы работы (максимум 500 символов)
- **phone** (string) - Телефон (максимум 50 символов)
- **website** (string) - Сайт (максимум 500 символов)

### 🔗 Связи с другими коллекциями
- **area** (relation: Many-to-One) - Район (из коллекции "Район")
- **categories** (relation: Many-to-Many) - Категории туризма (из коллекции "Категория")
- **tags** (relation: Many-to-Many) - Теги (из коллекции "Тег")

### ⚙️ Системные поля
- **is_active** (boolean) - Активно ли место (по умолчанию: true)
- **publishedAt** (datetime) - Дата публикации (draft/publish система)

## Как использовать в админ-панели

### 1. Создание нового места

1. Откройте http://localhost:1337/admin
2. В боковом меню выберите **"Место"**
3. Нажмите **"Create new entry"**
4. Заполните поля:
   - **Название** (обязательно)
   - **Фото** - загрузите изображение
   - **История** - добавьте описание с форматированием
   - **Адрес, Широта, Долгота** - укажите местоположение
   - **Часы работы, Телефон, Сайт** - контактная информация
   - **Район** - выберите из выпадающего списка
   - **Категории** - выберите одну или несколько
   - **Теги** - выберите подходящие теги
5. Нажмите **"Save"**
6. Нажмите **"Publish"** для публикации

### 2. Связи между коллекциями

#### Район (Many-to-One)
- Каждое место принадлежит **одному** району
- В выпадающем списке выберите район из 10 доступных

#### Категории (Many-to-Many)
- Место может иметь **несколько** категорий
- Выберите все подходящие категории туризма

#### Теги (Many-to-Many)
- Место может иметь **несколько** тегов
- Выберите теги, описывающие природные объекты

## API Endpoints

### Получить все места
```
GET /api/places?populate=*
```

### Получить место по ID
```
GET /api/places/:id?populate=*
```

### Фильтрация по району
```
GET /api/places?filters[area][id][$eq]=1&populate=*
```

### Фильтрация по категории
```
GET /api/places?filters[categories][id][$eq]=1&populate=*
```

### Фильтрация по тегу
```
GET /api/places?filters[tags][id][$eq]=1&populate=*
```

## 🧪 Что проверить сейчас

### Шаг 1: Откройте админ-панель
```
http://localhost:1337/admin
```

### Шаг 2: Проверьте Content-Type Builder
1. Откройте **Content-Type Builder**
2. Проверьте модель **Route** - должны быть все поля:
   - name, description, overview, history
   - type, area, image
   - stops (компонент)
   - distance_km, duration_hours, places_count
   - reviews, categories, tags
   - is_active

3. Проверьте компонент **route.route-stop**:
   - place (relation oneToOne)
   - order (integer)

4. Проверьте модель **Place** - должно быть новое поле:
   - entry (text)

### Шаг 3: Создайте тестовый маршрут

1. Перейдите в **Content Manager → Routes**
2. Нажмите **+ Create new entry**
3. Заполните:
   - **Name:** "Тестовый маршрут"
   - **Description:** "Описание для теста"
   - **Type:** выберите любой тип
   - **Area:** выберите любой район

4. Добавьте **Stops** (минимум 2):
   - Первая остановка: выберите место, order = 0
   - Вторая остановка: выберите другое место, order = 1

5. Нажмите **Save**

### Шаг 4: Проверьте автоматические расчёты

После сохранения маршрута проверьте, что автоматически заполнились:
- **distance_km** - расстояние между местами
- **duration_hours** - время прохождения
- **places_count** - количество мест (должно быть 2)

## ⚠️ Возможные ошибки при тестировании

### Ошибка: "Маршрут должен содержать минимум 2 места"
**Причина:** Добавлена только 1 остановка  
**Решение:** Добавьте минимум 2 остановки

### Ошибка: "В маршруте не должно быть дублирующихся мест"
**Причина:** Одно место добавлено дважды  
**Решение:** Используйте разные места для каждой остановки

### Ошибка: "У следующих мест отсутствуют координаты"
**Причина:** У места нет latitude или longitude  
**Решение:** Откройте место и заполните координаты

## 🔄 API запросы для тестирования

### Получить все маршруты
```bash
curl http://localhost:1337/api/routes?populate=*
```

### Получить маршрут с остановками
```bash
curl http://localhost:1337/api/routes/1?populate=stops.place,type,area,image
```

### Создать маршрут через API
```bash
curl -X POST http://localhost:1337/api/routes \
  -H "Content-Type: application/json" \
  -d '{
    "data": {
      "name": "API Маршрут",
      "description": "Создан через API",
      "type": 1,
      "area": 1,
      "stops": [
        {"place": 1, "order": 0},
        {"place": 2, "order": 1}
      ]
    }
  }'
```

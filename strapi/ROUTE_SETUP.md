# 🧭 Настройка создания маршрутов в Strapi

## ✅ Что уже настроено:

### 1. Схема маршрута (`route/schema.json`)
- ✅ Поле `places` - связь many-to-many с местами (POI)
- ✅ Поле `distance_km` - автоматически рассчитываемое расстояние (км)
- ✅ Поле `duration_hours` - автоматически рассчитываемое время (часы)
- ✅ Поле `places_count` - автоматически рассчитываемое количество мест
- ✅ Поля `distance` и `duration` для обратной совместимости

### 2. Lifecycle hooks (`route/lifecycles.js`)
- ✅ Автоматический расчет расстояния между местами (формула Хаверсина)
- ✅ Автоматический расчет времени прохождения (средняя скорость 3 км/ч)
- ✅ Автоматический подсчет количества мест
- ✅ Сохранение порядка мест из массива `places`
- ✅ Обработка ошибок и валидация координат

### 3. Связь с местами (`place/schema.json`)
- ✅ Добавлена обратная связь `routes` в модель Place

---

## 📋 Процесс создания маршрута в админке:

1. Открыть **Content Manager → Routes**
2. Нажать **"Create new entry"**
3. Заполнить:
   - **Name** (название) - обязательно
   - **Description** (описание) - обязательно
   - **Overview** (краткое описание) - опционально
   - **History** (история) - опционально
   - **Type** (тип маршрута) - выбрать из списка
   - **Area** (регион) - выбрать из списка
   - **Categories** (категории) - опционально
   - **Places** (места) - выбрать места **в нужном порядке**

4. Нажать **"Save"**

5. ✅ Автоматически рассчитываются:
   - `distance_km` - расстояние между местами
   - `duration_hours` - время прохождения
   - `places_count` - количество мест

---

## ⚙️ Как работает автоматический расчет:

### Формула Хаверсина:
Расстояние рассчитывается между последовательными точками маршрута по формуле Хаверсина (расстояние по дуге на сфере).

### Средняя скорость:
По умолчанию используется скорость **3 км/ч** (пеший туризм). Это можно изменить в файле `lifecycles.js`:

```javascript
const averageSpeed = 3; // км/ч
```

### Порядок мест:
Порядок мест в массиве `places` сохраняется. Расстояние считается между последовательными точками:
- Место 1 → Место 2
- Место 2 → Место 3
- Место 3 → Место 4
- и т.д.

## 🔧 Дополнительные настройки:

### Изменение средней скорости:
Откройте `/back/strapi/src/api/route/content-types/route/lifecycles.js` и измените:

```javascript
const averageSpeed = 4; // км/ч (для более быстрого темпа)
// или
const averageSpeed = 2.5; // км/ч (для медленного темпа)
```

### Настройка в зависимости от типа маршрута:
Можно добавить логику расчета скорости в зависимости от типа маршрута:

```javascript
// Пример: разные скорости для разных типов
const routeType = data.type_id;
let averageSpeed = 3; // по умолчанию

if (routeType === 1) { // Пеший поход
  averageSpeed = 3;
} else if (routeType === 2) { // Автомобильный
  averageSpeed = 50; // км/ч
} else if (routeType === 3) { // Велосипедный
  averageSpeed = 15; // км/ч
}
```

## ⚠️ Важные замечания:

1. **Порядок мест важен!** Порядок, в котором вы выбираете места в поле `Places`, определяет порядок прохождения маршрута.

2. **Координаты обязательны!** У всех мест должны быть заполнены поля `latitude` и `longitude`, иначе расстояние не будет рассчитано.

3. **Автоматический пересчет:** При обновлении маршрута и изменении списка мест, все данные пересчитываются автоматически.

4. **Read-only поля:** Поля `distance_km`, `duration_hours`, `places_count` автоматически заполняются и не должны редактироваться вручную (но в Strapi UI они будут редактируемыми - это ограничение Strapi).

## 📝 Пример создания маршрута через API:

```javascript
// POST /api/routes
{
  "data": {
    "name": "Маршрут по горным озерам",
    "description": "Красивый маршрут через несколько озер",
    "type": 1,
    "area": 1,
    "places": [1, 5, 12, 8] // ID мест в нужном порядке
  }
}
```

После сохранения автоматически рассчитаются:
- `distance_km`: 15.7
- `duration_hours`: 5.2
- `places_count`: 4

## 🐛 Устранение неполадок:

### Расстояние не рассчитывается:
1. Проверьте, что у всех мест заполнены `latitude` и `longitude`
2. Проверьте консоль Strapi на наличие ошибок
3. Убедитесь, что файл `lifecycles.js` сохранен и Strapi перезапущен

### Время не рассчитывается:
- Проверьте, что рассчитано `distance_km`
- Проверьте значение `averageSpeed` в `lifecycles.js`

### Порядок мест не сохраняется:
- Убедитесь, что вы выбираете места в правильном порядке в UI
- Проверьте, что массив `places` содержит ID в нужном порядке
